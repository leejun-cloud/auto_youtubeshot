# ⚡ YouTube Shots Creator (C형 포토 카드뉴스 제작기) 사용 설명서

본 프로그램은 초보자도 더블클릭 한 번으로 실행하여, AI 이미지 배경과 나레이션 목소리가 가미된 숏폼 비디오(유튜브 쇼츠 / 인스타그램 릴스)를 10초 만에 만들 수 있도록 설계된 로컬 웹 애플리케이션입니다.

---

## 🚀 1단계: 프로그램 시작하기 (원클릭 실행)

터미널이나 복잡한 명령어 입력 없이, 파일 더블클릭으로 즉시 시작할 수 있습니다.

### 🍎 Mac 사용자
1. 다운로드한 프로젝트 폴더 내 **`start.command`** 파일을 더블클릭합니다.
2. 검은색 터미널 창이 열리면서 필요한 프로그램들이 자동으로 설치 및 구동됩니다 (최초 1회 실행 시 약 1~2분 소요).
3. 가동이 완료되면 크롬 등 기본 인터넷 브라우저에서 작업 화면(`http://localhost:3000`)이 **자동으로 열립니다**.
   > **주의**: 프로그램 사용 중에는 터미널 창을 닫지 마세요. 터미널 창을 닫으면 프로그램이 종료됩니다.

### 💻 Windows 사용자
1. 프로젝트 폴더 내 **`start.bat`** 파일을 더블클릭합니다.
2. 검은색 도스(CMD) 창이 실행되며 자동 설치가 시작됩니다.
3. 구동이 완료되면 인터넷 브라우저가 열리며 자동으로 구동됩니다.

---

## 🔑 2단계: API 환경 설정 (최초 1회)

화면 상단의 **"API 환경 설정"** 카드에 필요한 API 키를 입력하세요. 한 번 입력해 두면 사용자의 브라우저에 안전하게 저장되어 다음 접속 시 다시 입력할 필요가 없습니다.

1. **Gemini API 키 (필수)**
   - 대본 기획, 성우 TTS 나레이션 음성 합성에 사용됩니다.
   - **발급 방법**: 입력창 아래 [Google AI Studio에서 무료 발급] 링크를 클릭한 후, 로그인하여 `Create API Key` 버튼을 눌러 발급받은 뒤 복사하여 붙여넣으세요.
2. **Pixabay API 키 / Pexels API 키 (선택)**
   - 테마에 어울리는 고품질 슬라이드 배경용 이미지/영상을 무료로 검색 및 다운로드하는 데 사용됩니다.
   - 각 발급 링크를 통해 10초 만에 무료 API 키를 얻어 입력해 두면 그라디언트 배경 대신 역동적인 이미지 배경이 자동으로 적용됩니다.

## 📝 3단계: 숏폼 영상 제작하기

### 1. 스크립트 작성
- **방법 A (AI 자동 대본 작성)**:
  `[간단한 아이디어/주제 입력]` 란에 만들고 싶은 주제를 적으세요.
  *예: "마음을 차분하게 만드는 3가지 명상법", "부자가 되기 위해 버려야 할 나쁜 습관"*
  이후 우측의 **`[AI 대본 짜기]`** 버튼을 누르면 Gemini가 카드뉴스 전용 규격(제목, 본문 카드 3~4장, CTA, 전체 나레이션)으로 대본을 즉시 작성해 줍니다.
- **방법 B (직접 입력)**:
  원하는 대본 형식이 있다면 텍스트 영역에 직접 본문을 입력하거나 수정할 수 있습니다.

### 2. 비디오 생성 시작
- 본문 텍스트가 모두 완성되었다면 최하단의 **`[🎬 9:16 비디오 생성 시작]`** 버튼을 클릭합니다.
- 우측 패널에 진행률 프로그레스 바가 작동하며 단계별 작업 상황(텍스트 구조화 ➔ 미디어 다운로드 ➔ 나레이션 TTS 합성 ➔ Remotion 영상 렌더링)을 시각적으로 모니터링할 수 있습니다.

---

## 📺 4단계: 결과 다운로드 및 업로드 정보 복사

제작이 완료(100%)되면 화면 우측 하단에 비디오 재생 화면과 복사 패널이 나타납니다.

1. **비디오 플레이어**: 폰 프레임 내부에서 렌더링된 비디오를 재생해 보고 싱크를 확인합니다.
2. **MP4 다운로드**: `[💾 MP4 동영상 다운로드]` 버튼을 클릭해 컴퓨터에 영상 파일을 소장합니다.
3. **업로드 정보 복사 (마케팅 도구)**:
   - **유튜브 쇼츠 제목**: 조회수를 폭발시키기 위해 Gemini가 작성한 바이럴 후킹 제목입니다. `[복사]`를 눌러 유튜브 쇼츠 업로드 시 붙여넣으세요.
   - **유튜브 쇼츠 설명 & 태그**: 해시태그와 함께 잘 쓰인 소개글입니다. `[복사]`를 눌러 쇼츠 상세설명란에 넣으세요.
   - **인스타그램 Reels 캡션**: 릴스 알고리즘에 도달하기 좋은 긴 공감형 캡션과 25~30개의 전략적 해시태그가 통합된 글입니다. `[복사]`를 눌러 인스타그램 본문에 입력하세요.

## 💡 자주 묻는 질문 (FAQ)

**Q. 영상 생성 중 오류가 나거나 너무 오래 걸려요.**
A. 로컬 컴퓨터의 리소스 상황이나 네트워크 속도에 따라 배경 미디어 다운로드 및 비디오 합성 속도가 다를 수 있습니다. 특히 최초 렌더링 시에는 Remotion 컴파일러 빌드가 시작되므로 약 30~50초가 추가로 소요됩니다. 2번째 영상 생성부터는 속도가 눈에 띄게 빨라집니다.

**Q. FFmpeg 연결 상태가 '미설치'로 나옵니다.**
A. `package.json` 의존성에 의해 원클릭 구동 시 백그라운드에서 자동으로 로컬 FFmpeg 환경이 구성됩니다. 만약 네트워크 문제 등으로 환경 구성이 누락된 경우, 우측 상단 상태 표시줄의 `[자가 설치]` 버튼을 누르면 프로젝트 폴더 내로 강제 구성하여 즉시 해결할 수 있습니다.
